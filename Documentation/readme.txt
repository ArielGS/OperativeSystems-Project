	<Intrucciones:>

		1. Instalación de grpc:
			- python.exe -m pip install --upgrade pip
			- python -m pip install grpcio
			- pip install grpcio grpcio-tools
		
		2. Se abre la carpeta 'OperativeSystems-Project' en VisualStudio Code
	
		3. Se selecciona el archivo Server.py y se ejecuta sin debuguear.
	
		4. Se selecciona el archivo Application.py y se ejecuta sin debuguear.

	<Consideraciones:>
		
		1. Esta proyecto es en consola, no cuenta con GUI.

		2. El proyecto cuenta con persistencia, cada archivo está guardado en su
		   respectiva carpeta.

		3. En la carpeta "Documentación" se adjunta un pdf con capturas del
		   funcionamiento del proyecto además de las referencias bibliográficas.
